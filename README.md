## 🇬🇧 English

**SeaRL - Relámpago Negro** is an independent, open-source video and audio downloader for Android. It is a fork of the original [Seal](https://github.com/JunkFood02/Seal) application, designed to provide a more streamlined bulk downloading experience with absolutely no content restrictions and a beautiful pure black and neon red theme.

### 📖 Key Features
- **Bulk URL Download**: Paste and download multiple links sequentially at once.
- **Smart Delay Queue**: Sequential queue downloads with an **automatic 20-second wait time** between files to prevent IP rate-limiting and bans.
- **Instagram Story & Carousel Downloader**: Download photos, posts, reels, and full stories.
- **Interactive Selection Dialog**: Choose exactly which slides/stories to download from carousels or user story feeds, with the option to download all at once.
- **Integrated WebView Login Browser**: Log in securely and easily to Instagram, Threads, and X/Twitter within the app to automatically capture cookies, bypassing the need for manual netscape `cookies.txt` imports for non-technical users.
- **Direct High-Quality Media Downloads**: Natively downloads Instagram and Threads media via OkHttp for faster, robust downloads without running into file name length errors.
- **No Content Restrictions**: Unrestricted downloads—SeaRL does not block any content (unlike the original app).
- **Audio-Only Mode**: Quickly download only the audio track from videos.
- **Fast File Sharing**: Share downloaded files instantly via the built-in share option.
- **Locate Files Instantly**: Directly open the download folder or locate the file on your device from the download list.
- **Modern UI & Aesthetic**: Pure black backgrounds (`#000000`) paired with vibrant neon red accents (`#E11D48`) for a premium force dark-mode experience.
- **Platform Badges**: Automatic recognition and badges for YouTube, Threads, Instagram, and X/Twitter.
- **yt-dlp Powered**: Inherits support for thousands of video sites via yt-dlp.
- **Multilingual Support**: Fully translated into multiple popular languages.

## 🇨🇳 简体中文

**SeaRL - Relámpago Negro** 是一款独立、开源的 Android 视频和音频下载工具。SeaRL 派生自 [JunkFood02](https://github.com/JunkFood02) 开发的原版 [Seal](https://github.com/JunkFood02/Seal) 应用，旨在提供更高效的批量下载体验，没有任何内容审查和限制，采用持久下载队列，并搭配排版精美的纯黑与霓虹红主题。

### 📖 核心功能
- **批量 URL 下载**：一次性粘贴并按顺序下载多个链接。
- **智能延迟队列**：下载任务按顺序在队列中执行，并在任务之间加入 **20秒的自动等待时间**，有效防止因频繁请求被平台封锁 IP 或触发速率限制。
- **Instagram 故事与轮播下载器**：支持下载 Instagram 的照片、帖子、Reels 和完整的用户故事（Stories）。
- **交互式媒体选择对话框**：在轮播帖子或多条故事中，自由勾选需要下载的照片/视频，或一键下载全部。
- **内置 WebView 登录浏览器**：直接在应用内安全、便捷地登录 Instagram、Threads 和 X/Twitter，自动捕获所需的 Cookie，免去非技术用户手动导入 `cookies.txt` 的繁琐操作。
- **高品质直接下载**：使用 `OkHttp` 原生下载 Instagram 和 Threads 媒体，下载更快速稳定，有效避免长文件名保存出错的问题。
- **无内容限制**：无限制下载——SeaRL 不会屏蔽任何内容（与原版应用不同）。
- **仅下载音频**：快速提取视频的音频轨道。
- **快速分享文件**：通过内置的分享选项立即分享已下载的文件。
- **快速定位文件**：直接在下载列表中打开下载文件夹或定位设备上的文件。
- **现代视觉设计**：纯黑背景（`#000000`）搭配动感的霓虹红点缀（`#E11D48`），提供极致的暗黑模式体验。
- **平台专属徽章**：自动识别并显示 YouTube、Threads、Instagram 和 X/Twitter 的徽章。
- **yt-dlp 强力驱动**：继承 yt-dlp 对数千个视频网站的强大支持。
- **多语言支持**：已完整翻译为多种常用语言。
